Mail Service

Simply messaging, send request & send message

How it works:

  • Rename .env.example to .env
  • Fill .env file with your e-mail account data
  • Send request using POST for http://your-domain.example/send

Example payload:

{
"message": "Your message content",
"title": "Message to you!",
"recipientMail": "recipient@mail.example",
"ccMail": "copy@mail.example",
"bccMail": "hidden-copy@mail.example"
}

Also, you can set multiple recipients, add CC (carbon copy) or BCC (blind carbon copy) recipients like this:

{
"message": "Your message content",
"title": "Message to you!",
"recipientMail": ["recipient@mail.example", "another-recipient@mail.example"],
"ccMail": ["copy@mail.example", "another-copy@mail.example"],
"bccMail": ["hidden-copy@mail.example", "another-hidden@mail.example"]
}
